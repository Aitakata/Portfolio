Projeto Seetec

App Secretaria Escolar Digital

Projeto desenvolvido durante o curso de desenvolvimento móbile da ETEC Sebrae (Campos Elísios) do Centro Paula Souza para Interação entre os alunos e a Secretaria da escola para solicitação e acompanhamento de documentos, bem como para obter informações. O Front-End App foi desenvolvido em Flutter, o acesso na secretaria (desktop) em a parte de Back-End em Java, para interface foram utilizadas API Rest.

Projeto Pitu

URL Shortener

O Pitu é single micro-SaaS na cloud para encurtamento de URLs. Permite perrsonalizar os seus links, encurtandos e criando endereços para a sua marca. Desta forma, os usuários terão mais facilidades no reconhecimento do seu negócio em meio a tanta concorrência no ambiente digital.

Projeto Netflix - Dio

Recriando a interface Netflix

Projeto desevolvido em Javascript/Typscript durante o curso Dio. O objetivo foi recriar a interface de comunicação como o Netflix. A ideia é ter um base de dados com informações básicos dos filmes disponíveis: Uma resenha do filme, uma imagem, diretor, elenco, etc. Atráves do App é possível acessar os dados e programar a visualização do filme.
